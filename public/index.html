The Traffic Flow Predictor is a machine learning-powered web application designed to forecast traffic volume based on environmental and temporal inputs such as temperature, rainfall, weather conditions, date, and time. Built with Python and scikit-learn for model training and prediction, and Node.js for backend integration, the project demonstrates how AI can be used to make informed decisions in urban planning and travel optimization.

  • Interactive web interface for real-time predictions

  • Efficient model scaling using MinMaxScaler
  • Deployed with Node.js server integration
